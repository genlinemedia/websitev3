# Genline - Creative Systems for Ad Agencies

**Stop ad fatigue and client churn.** Genline treats creative messaging like math—data-driven creatives that scale, not clone.

## Project Overview

A modern, responsive landing page built with React, Vite, TypeScript, TailwindCSS, and Framer Motion animations. Features scroll-reveal animations, mobile-optimized design, and a sleek dark theme.

## Tech Stack

- **Frontend Framework**: React 19
- **Build Tool**: Vite 6
- **Language**: TypeScript
- **Styling**: TailwindCSS 3.4
- **Animations**: Framer Motion 12.36
- **Icons**: Lucide React
- **CSS Processing**: PostCSS & Autoprefixer

## Features

✨ **Scroll-Reveal Animations** - Sections fade in and slide up as they enter the viewport  
📱 **Fully Responsive** - Optimized for mobile, tablet, and desktop  
🎨 **Modern Design** - Dark theme with orange accents and orbital elements  
⚡ **Performance Optimized** - Fast load times with proper optimization  
🎥 **Video Integration** - Embedded VSL with custom thumbnail support  
📧 **Form Handling** - Formspree integration for lead capture  
♿ **Accessible** - Semantic HTML and proper ARIA labels  

## Getting Started

### Prerequisites
- Node.js 16+ and pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/genline.git
cd genline

# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm run build
```
